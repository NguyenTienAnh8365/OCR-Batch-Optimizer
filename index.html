Config & Benchmark Data

Cài đặt tham số hệ thống và nhập dữ liệu đo hiệu năng
⚙️ Tham số hệ thống / System Parameters
warmup_runs — Số run khởi động ?
m_max_extrap — Hệ số nội suy tối đa ?
m_max_concurrent — Giới hạn GPU (VRAM) ?
n_sim — Số mô phỏng Monte Carlo ?

§ 02

Benchmark Analysis POST /benchmark

Phân tích dữ liệu đo thực, khớp mô hình Power Law, xem thống kê từng batch size
📊
Bấm nút trên để phân tích dữ liệu đã nhập
§ 03

Find Optimal m* POST /optimize

Tìm số workers tối ưu để xử lý N requests đang chờ trong hàng đợi
N — Số request đồng thời ?
mode — Chiến lược xử lý ?
latency metric — Cách tính latency worst-case ?
🎯
Chọn N và mode rồi bấm Tìm m*
§ 04

Bulk Optimize POST /optimize/bulk

Tối ưu nhiều kịch bản N cùng lúc — dùng để vẽ đồ thị m* theo N
N_list — Danh sách số request (cách nhau dấu phẩy) ?
mode— Chiến lược
latency metric— Cách tính
📦
Nhập danh sách N rồi bấm Chạy
§ 05

Batch vs Pool POST /compare

So sánh trực tiếp hai chiến lược xử lý cho cùng một N — xem chiến lược nào tốt hơn trong trường hợp của bạn
N— Số request cần so sánh
latency metric— Cách tính latency
⚖️
Bấm So sánh để thấy sự khác biệt giữa hai chiến lược

API Endpoints
